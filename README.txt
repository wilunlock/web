# Carpeta de Assets

Esta carpeta debe contener:

1. logo.jpeg - El logo principal del sitio
   - Tamaño recomendado: 200x200px o mayor
   - Formato: JPEG, PNG o SVG
   - Fondo transparente preferible

Para usar tu propio logo:
- Coloca tu archivo de logo en esta carpeta
- Nómbralo como "logo.jpeg" (o actualiza las referencias en el código)
- O usa el panel de administración para cambiar el logo desde una URL

El logo se usa en:
- Header del sitio
- Footer
- Favicon
- Panel de administración
